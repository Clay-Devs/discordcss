Discord.css in action

Why Discord.css?

Discord.css is entirely based on Water.css which allows you to make simpler webpages, like this one, look much better. We're taking this further by making Water.css look really close to Discord's CSS so non CSS developers can make Discord-related websites- like bot dashboards much easier. Discord.css is simple, easy, and fast.

    User's choice respecting

    Discord.css checks what the user displaying the webpage's pereference is on light/dark mode and adjusts the website to it with magic! The background color gets changed to Discord's light/dark mode color depending on the user's choice! This is happening right now on this website as well :D

    An image

    Discord.css will find every image in the webpage and round it so that it looks a little better :). You can stop this from happening by adding an !important CSS rule to the .img tag! See the rounded image below? Discord.css rounded it for you! This makes the image look much more modern and beautiful at most times.
